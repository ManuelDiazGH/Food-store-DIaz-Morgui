# Proposal: admin-product-crud

## What
Implementar las tareas pendientes del Sprint 3: frontend Admin/Stock CRUD de productos (formularios de creación/edición, gestión de stock, listado), actualización de navbar con links faltantes, y verificación de funcionamiento.

## Why
El Sprint 3 se completó parcialmente. Quedaron sin implementar las páginas de gestión de productos para los roles STOCK y ADMIN (tareas 9.1-9.7 de `tasks.md`), la actualización del navbar con links a catálogo y perfil (tarea 11.4), y las verificaciones finales (tareas 12.x).

## Scope
- Frontend: páginas de gestión de productos para STOCK/ADMIN
- Frontend: componentes reutilizables para formulario de producto, selector de categorías/ingredientes, gestor de stock
- Frontend: actualización de navbar con links de catálogo y perfil
- Verificación: build exitoso y funcionamiento correcto
